# FostercareRezTimer
A lightweight PvP respawn timer for **Turtle WoW** (1.12.1). It tracks the enemy Spirit Healer wave and counts enemy deaths.

## Commands
* `/frt show` - Forces the timer to appear (for testing/moving).
* `/frt hide` - Hides the timer.
* `/frt lock` - Locks or unlocks the frame.

## Installation
1. Download and extract to `Interface\AddOns\`.
2. Rename folder to `FostercareRezTimer`.
3. Restart game.
